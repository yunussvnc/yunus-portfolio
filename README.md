# Personal Portfolio Site

React + TypeScript + Tailwind CSS portfolio site for Yunus Emre Sevinc.

## Sections

- Home
- Projects
- About
- Contact

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deployment

The project is ready for static deployment on Vercel, Netlify, or GitHub Pages. For Vercel-style SPA hosting, `vercel.json` includes a rewrite to `index.html`.
